# 赛博朋克粒子交互系统开发指令

角色设定：你是一个精通 Three.js、WebGL 和计算机视觉（MediaPipe Hands）的前端创意开发专家。请编写一个单文件 HTML 应用。
视觉风格：
主题：硬核赛博朋克（Cyberpunk）。
背景：深黑色背景，带有移动的扫描线或网格背景图，以及暗角（Vignette）效果。
HUD：屏幕四角有极简风格的数据显示（FPS、粒子数、左右手状态），使用 Orbitron 或等宽字体，颜色为霓虹青色。
粒子材质：发光粒子，使用 AdditiveBlending。
核心参数：
粒子数量：12,000 个。
粒子大小：2.4。
物理手感：回弹速度快（0.16），斥力强且干脆。
移动端适配：不强制前置摄像头，界面自适应。
交互逻辑（详细需求）：
1. 左手（指令控制）： 检测左手伸出的手指数量，根据数量切换粒子目标形状和颜色：
1 指：显示文字 "Hello"，颜色：荧光蓝 (0x00FFFF)。
2 指：显示文字 "Gemini3"，颜色：荧光黄 (0xFFFF00)。
3 指：显示文字 "非常好用"，颜色：荧光粉 (0xFF00FF)。
4 指：显示文字 "再见"，颜色：荧光绿 (0x00FF88)。
5 指（全张开）：进入** “盛接模式” ** (Catch Mode)，作为粒子飞行的终点容器。
2. 右手（物理互动）：
追踪点：始终追踪食指指尖（Index Finger Tip）。
状态 A（普通/握拳/食指）：
当触碰粒子时，触发**强力散开（Scatter）**特效。
关键点：只要平面的推开效果，不要 Z 轴方向的凸起或畸变。
状态 B（全张开）：
触发星云模式：粒子不再组成文字，而是散布全屏。
交互：右手划过时产生**水波纹（Ripple）**效果。
3. 双手联动（大招特效）：
触发条件：当 右手张开（星云背景激活） 且 左手也张开（盛接姿态激活）时。
特效逻辑：
所有全屏粒子瞬间被吸引飞向 左手掌心。
形态：粒子在左手中汇聚成一颗旋转的篮球（使用斐波那契球面算法分布，橙色为主，带有模拟黑线的纹理）。
动态：粒子在飞入过程中要有弹跳轨迹（Jumping/Bouncing），像无数颗弹力珠跳进手里。
技术实现要求：
使用 Canvas 像素扫描法生成文字坐标（字号 75px）。
MediaPipe Hands 配置为检测 2 只手。
代码必须包含在一个完整的 index.html 文件中，内嵌 CSS 和 JS。
使用 requestAnimationFrame 进行流畅渲染。
