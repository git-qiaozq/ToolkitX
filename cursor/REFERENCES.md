## 核心功能

### 1. Agent - 将想法转化为代码

人机协作编程，比单独开发效率高出数个量级。Agent 可以：
- 理解你的意图并生成代码
- 执行多步骤复杂任务
- 在后台自主工作

### 2. Tab - 神奇的自动补全

自定义的 Tab 模型以惊人的速度和精度预测你的下一步操作：
- 减少 21% 的建议次数
- 提高 28% 的接受率

### 3. 生态系统集成

Cursor 无处不在：
- **GitHub**：审查你的 PR
- **Slack**：作为团队成员协作
- **BugBot**：自动发现和修复 Bug

### 4. 多模型支持

支持所有前沿模型：
- OpenAI (GPT-5)
- Anthropic (Claude Sonnet 4.5, Claude Opus 4.5)
- Google (Gemini 3 Pro)
- xAI (Grok Code)

### 5. 代码库理解

无论规模和复杂度如何，Cursor 都能学习你的代码库工作方式。
